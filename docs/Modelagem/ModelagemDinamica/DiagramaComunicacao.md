## Introdução

O diagrama de colaboração (ou de comunicação) representa como os objetos de um sistema interagem entre si, focando nos relacionamentos e trocas de mensagens, mais do que na ordem temporal dessas ações [APOSTILA UML]. Segundo a IBM [IBM], esse tipo de diagrama permite analisar o comportamento dinâmico do sistema, identificando objetos participantes, dados trocados e caminhos de mensagens. Como destaca o Guia UML [GUIA UML], ele funciona como um mapa visual que mostra a estrutura das interações internas. Na Galáxia Conectada, esse modelo é utilizado para representar de forma clara as comunicações entre os componentes da aba de conhecimento e do bot de promoções.
